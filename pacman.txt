# Arch Linux packages

# Package priority order:
#   1. Omarchy repo (custom repo, preferred source)
#   2. Official Arch repos (core, extra, multilib)
#   3. AUR (requires yay or paru)
#
# Installation:
#   Quick install all: grep -v '^#' pacman.txt | grep -v '^$' | xargs yay -S --needed
#   Check package: eden check-package <name>
#
# Note: yay/pacman automatically selects the best source in priority order
